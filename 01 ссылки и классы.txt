1.	Написать функцию, которая принимает по ссылке три переменных и записывает в них текущую дату. Использовать функции time и localtime библиотеки time.h
2.	Написать функцию, которая в качестве параметров принимает число N и некую переменную len (по ссылке). 
Функция возвращает массив целых чисел, в котором будут находиться все простые числа от 2 до N включительно. В переменную len записать длину массива.
3.	Есть структура, содержащая массив и его длину. Написать функцию getElem, которая возвращает ссылку на i-тый элемент массива 
(i передается как параметр). Написать функцию init, которая бы записывала в этот массив случайные числа без повторений. 
Для генерации случайных чисел использовать функцию rand.  Случайные числа должны быть не больше длины массива, умноженной на два. 
Для доступа к элементам массива использовать функцию getElem.
4.	Написать функцию f1, которая содержит локальную статическую переменную k, возводит ее в квадрат и возвращает ссылку на эту переменную. 
Написать функцию f2, которая счи-тывает число с клавиатуры и присваивает это значение переменной k функции f1.


5.	Разработать класс для работы со строками. Нужно поддерживать функции создания строки, создания экземпляра класса из типа char*, 
добавления подстроки в существующую строку на нужное место, функцию поиска подстроки в строке, возвращения подстроки. Предусмотреть контроль над памятью.
6.	Класс дата. Содержит конструкторы 
a.	По миллисекундам
b.	По году, числу, месяцу.
Содержит функции для работы с датами – получения дат, задания дат, вычисления разницы между двумя датами и т.д.
